# IGamesEngine

## 介绍

- IGamesEngine是学习[Games](https://games-cn.org/)系列课程的后实现的游戏引擎
- 使用`c-sharp`以面向接口方式实现。

## 编译要求

- [dotnet](https://dotnet.microsoft.com/zh-cn/)
- [PowerShell](https://github.com/PowerShell/PowerShell)

## 注意

- 这是一个个人项目，并不保证稳定更新。
- games系列作业版本将以`c`结尾
